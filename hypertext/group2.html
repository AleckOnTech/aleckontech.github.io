What is Hypertext and How can we create it?

Author(s): Alethea Joelle Nicol, Audrie Penecilla, Catherine Sotelo, Chris Aleck Bernal, Jade Chloe De Vera, John Lloyd Dela Cerna, Lindsay Rhian Sanchez, Neithan John Apostol, Robert Ruel IV and Ryza Charize Parane.
Published on: September 11, 2025.   NOTE: THIS IS THE FIRST DRAFT OF THE ARTICLE.

 In today's digital world, most individuals interact with hypertext whenever they use a computer or browse the internet. Hypertext, often referred to as “links,” allows users to move to another section of a document, a different webpage or website. Designed for easy access and organizes information in a way that is simple and user-friendly. This system had become essential for referencing, browsing websites, and connecting information from many different sources.

This article explains the definition and importance of hypertext and shows how it can be created using different tools, including HTML, Microsoft Word, Google Docs, and Canva.

What is Hypertext?

  Hypertext is a digital text format that contains links to other texts or resources, allowing for non-linear navigation through information (Wikipedia, 2025a; Haynes, 2023). Unlike traditional linear text, which is read sequentially, hypertext provides a web-like structure of interconnected information, enabling users to access relevant content immediately via clickable references. This interactivity forms the foundation of the modern World Wide Web (W3C, n.d.; Link-Assistant, n.d.).

  The primary characteristic of hypertext is its embedded links, known as hyperlinks, which connect text to additional information either within the same document or across multiple documents (MDN Web Docs, 2025a). These hyperlinks facilitate a dynamic reading experience, allowing users to explore related topics at their own pace rather than following a predetermined sequence. Thus, hypertext is not merely text but a framework for organizing and accessing knowledge interactively (Wikipedia, 2025b).

  Hypertext can encompass various forms of content, including text, images, and multimedia elements, all interconnected through hyperlinks (MDN Web Docs, 2025b). The distinction between hypertext and hyperlinks is critical for understanding web-based information structures. Hypertext refers to the text itself that contains links, whereas a hyperlink is the actual clickable reference that connects one document to another (Wikipedia, 2025a; MDN Web Docs, 2025a). Together, they form the backbone of interactive and navigable digital documents.

Why is it used?

  Hypertext is used primarily to facilitate non-linear access to information, allowing users to navigate quickly between related topics or documents (Wikipedia, 2025b; W3C, n.d.). Unlike traditional printed text, which requires sequential reading, hypertext enables a dynamic and interactive exploration of content, improving efficiency in learning, research, and information retrieval (Haynes, 2023).

  Hypertext also supports the interconnection of diverse content types, such as text, images, videos, and other multimedia, creating a rich, multi-dimensional information environment (MDN Web Docs, 2025b). By embedding hyperlinks within text, hypertext allows users to access supporting materials, definitions, or related topics immediately, reducing the need to search manually for additional resources.

  Moreover, hypertext underpins the functionality of the World Wide Web, enabling websites to form a connected network of information. This interconnected structure supports knowledge sharing, collaborative learning, and efficient navigation across large datasets, making it an essential tool for digital communication, education, and research (Link-Assistant, n.d.; MDN Web Docs, 2025a).

How is Hypertext created?

 Hypertext can be created in differnet places, but this article focuses on four common tools: Hypertext Markup Language (HTML), Microsoft Word, Google Docs and Canva.

On Hypertext Markup Language (HTML)

In HTML, hyperlinks is created by wrapping the text or other content inside an <a> (anchor) tag, with the href attribute specifying the destination URL or section within the same page. (MDN Web Docs, 2025b)

Example of a basic link

<a href="http://google.com">Go to Google!</a>

Example of an Internal Bookmark Link

<a href="#section1">Go to Section 1</a>
<p id="section1">Content of Section 1</p>

Note that links are not limited to just plain text. Here is an example of a link using within a header.

<a href="https://facebook.com">
 <h2>https://facebook.com"</h2>
</a>

Here is the output:

http://facebook.com

On Microsoft Word

In Microsoft Word, hyperlinks can be added to text, images, or shapes.

  1. Highlight the text or select the object to hyperlink.
  2. Right-click and choose Link, or press Ctrl + K.
  3. Enter the URL or select a location within the document.
  4. Click OK to apply the hyperlink.

On Google Docs

In Google Docs, hyperlinks are added in a similar manner.

  • Highlight the text you want to link.
  • Click Insert > Link, or press Ctrl + K.
  • Enter the URL or select a heading/bookmark within the document.
  • Click Apply.

On Canva

In Canva, hyperlinks can be added to text, images, or shapes in a design.

Steps:

  1. Select the element to hyperlink.
  2. Click the Link icon in the toolbar.
  3. Enter the URL or select a page within the design.
  4. Click Apply.

Conclusion

  Hypertext is a foundational component of the digital information ecosystem, enabling non-linear navigation and interactive access to content. By embedding hyperlinks within text, hypertext allows users to efficiently explore related documents, multimedia, and web resources, creating a dynamic and interconnected network of information (Wikipedia, 2025b; MDN Web Docs, 2025a). Its use enhances knowledge accessibility, supports learning and research, and underpins the functionality of the World Wide Web (W3C, n.d.; Haynes, 2023).

  Creating hypertext involves structuring content so that it contains clearly defined links to relevant resources, whether through text, images, or other digital media (MDN Web Docs, 2025b; Link-Assistant, n.d.). The process emphasizes clarity, accessibility, and logical organization, ensuring that users can navigate information intuitively.

  In conclusion, hypertext not only revolutionizes how information is consumed but also facilitates a flexible and user-directed approach to learning and exploration. Understanding both the concept of hypertext and the methods for creating it is essential for anyone seeking to engage effectively with digital content and the modern web.
